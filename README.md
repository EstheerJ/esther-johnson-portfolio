# Esther Johnson Portfolio

Static portfolio site for Esther Johnson, Content Designer & UX Writer.

## Preview Locally

Open `index.html` in a browser.

## Publish

Publish only this `portfolio-site` folder. Do not publish the parent workspace because it contains planning drafts and confidential working materials.

Recommended options:

- Netlify drag-and-drop: upload this folder.
- GitHub Pages: push this folder as the site root.
- Vercel/Netlify from GitHub: set this folder as the deployed project.
